# repair-ui-copy

A Codex skill for fixing broken UI copy, garbled Chinese text, and layout issues caused by long labels.

## What it captures
- Source-level repair for corrupted strings
- Safer editing habits when PowerShell and WSL are both involved
- Overflow guards for cards, dialogs, and badges
- Short, explicit toast copy

## Install
1. Clone or download this repository.
2. Copy the repository folder into `~/.codex/skills/repair-ui-copy`.
3. Start a new Codex session.
